Github Pages branch of worldbuilding-docs, containing sources for the github.io/Jekyll site at http://torbenbarsballe.github.io/worldbuilding-docs/

Building Locally:

* Install [Ruby](https://www.ruby-lang.org/en/downloads/)
* Install the the gem `bundle` and run `bundle install`:
  ```
  gem install bundle
  bundle install
  ```
* Run `bundle exec jekyll serve`

update-sources.sh is used to pull the latest changes from master and add Jekyll headers.
